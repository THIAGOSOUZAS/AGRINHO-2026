# AGRINHO-2026
Descrição
Este é um site moderno, responsivo e otimizado para apresentar serviços de desenvolvimento web. O projeto inclui uma página inicial (hero), seção de serviços e footer simples.
O site foi desenvolvido como projeto escolar por Thiago de Souza, aluno da 3ª Série A do Colégio Estadual Marechal Costa e Silva.
Tecnologias Utilizadas

HTML5
CSS3 (arquivo style.css)
JavaScript (arquivo script.js)

Estrutura do Projeto
text/
├── index.html
├── style.css
├── script.js
└── README.md
Como Executar

Baixe os arquivos do projeto.
Abra o arquivo index.html em qualquer navegador moderno.
(Opcional) Para desenvolvimento, utilize uma extensão como "Live Server" no VS Code.

Funcionalidades

Design responsivo
Navegação suave
Seção hero atrativa
Cards de serviços
Botões de chamada para ação
